# mengyue-components

`mengyue-components` 是本组件库的 **正式版本**。它在继承早期过渡版全部组件能力的基础上，完成了 API 的统一化设计与质量打磨，是当前推荐的生产级组件解决方案。

## 核心特性

- **Vue 3 + TypeScript**：充分利用 Composition API 与类型推导，提供完整的 TS 类型定义。
- **无第三方依赖**：所有组件均为自研实现，样式与逻辑高度可控，避免外部库的版本捆绑。
- **100% 测试覆盖**：单元测试用例超过 200 个，确保每个组件的功能边界与交互行为稳定可靠。
- **轻量级打包**：全量引入体积不足 1MB，且天然支持按需加载，对项目体积几乎无负担。

## 与早期版本的区别

相比于过渡版本 `mengyue-plus`，`mengyue-components` 进行了以下关键改进：

- **统一的 Props API**：重新梳理并规范了所有组件的属性命名与使用方式，降低学习与记忆成本。
- **缺陷修复**：集中处理了早期版本中积累的已知问题，代码健壮性进一步提升。
- **长期维护保障**：作为正式版本，将持续接收功能迭代、性能优化与社区反馈修复。

> 功能层面二者完全对等，**正式版是对早期版本的彻底完善与规范化**。

## 适用场景

`mengyue-components` 适用于所有基于 Vue 3 + TypeScript 构建的中后台系统、工具型产品及轻量化 Web 应用。其轻量、纯净的设计理念尤其适合对代码体积或定制自由度有较高要求的项目。
