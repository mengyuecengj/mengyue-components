# v0.0.30 Release
This version represents the 0.0.30 release of the component library. In this version, layout and scroll components were implemented and unit tested using `Vitest`. Implemented in Vue files, this marks the 30th release to date, classified as an early testing version.

## Current Library Components

### Basic Components
- Button
- Border
- a
- text

### Layout and Scroll Components
- scroll (global)
- scrollbar (local)
- container
- layout
