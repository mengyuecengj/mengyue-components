# v0.0.40 Release
This version represents the 0.0.40 release of the component library. In this version, form components were initially implemented and unit tested using `Vitest`. Implemented in Vue files, this marks the 40th release to date, classified as an early testing version. The current form implementation still has some potential issues, hence its classification as a testing version. This also represents a transitional phase for form components.

- Border
