# v0.0.68-beta.1 版本
该版本为组件库的0.0.68-beta.1版本。在此版本中完成了测试版本演进的全部组件测试，为测试版本演进到正式版本的过度版本，为第一个正式版本发布前的第一个测试版，到次将组件库命名从`mengyue-plus`改成了`mengyue-components`，到此处已经发布了68次，正式版本的第一次发布。
This version represents the 0.0.68-beta.1 release of the component library. In this version, all components from the testing phase evolution have been thoroughly tested, serving as a transitional release between the testing versions and the official release. This is the first beta release prior to the official version launch. At this point, the library name was changed from `mengyue-plus` to `mengyue-components`. This marks the 68th release to date and represents the first official release.

## Installing the Component Library
The installation method needs to be updated from this point forward:

```bash 
pnpm install mengyue-components
npm install mengyue-components

// Import in your project main
import MengyueComponents from 'mengyue-components'
import 'mengyue-components/style.css'

const app = createApp(App);
app.use(MengyueComponents)
app.mount('#app');
```
