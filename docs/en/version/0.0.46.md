# v0.0.46 Release
This version represents the 0.0.46 release of the component library. In this version, form components were initially implemented and unit tested using `Vitest`. Implemented in Vue files, this marks the 46th release to date, classified as an early testing version. The current version has completed testing and represents the first usable form component release.
