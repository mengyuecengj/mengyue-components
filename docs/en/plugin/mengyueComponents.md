# mengyue-components

`mengyue-components` is the **official version** of this component library. Building upon the full capabilities of the early transitional version, it features a unified API design and refined quality, making it the recommended production-grade component solution.

## Core Features

- **Vue 3 + TypeScript**: Leverage the Composition API and type inference to provide complete TS type definitions.
- **No Third-Party Dependencies**: All components are self-developed and implemented, with highly controllable styles and logic, avoiding version bundling of external libraries.
- **100% Test Coverage**: With over 200 unit test cases, ensuring the functional boundaries and interaction behaviors of each component are stable and reliable.
- **Lightweight Packaging**: The full import size is less than 1MB, and it natively supports on-demand loading, posing almost no burden on project size.

Differences from earlier versions

Compared to the transitional version `mengyue-plus`, `mengyue-components` has made the following key improvements:

- **Unified Props API**: Streamlined and standardized the naming and usage of all component properties to reduce learning and memorization costs.
- **Defect Fixing**: Known issues accumulated in early versions were addressed in a concentrated manner, further enhancing code robustness.
- **Long-term Maintenance and Support**: As an official release, it will continue to receive feature iterations, performance optimizations, and community feedback fixes.

On the functional level, the two are completely equivalent, with the **official version representing a thorough refinement and standardization of the early version**.

## Applicable Scenarios

`mengyue-components` is suitable for all backend systems, tool-type products, and lightweight web applications built with Vue 3 + TypeScript. Its lightweight and minimalist design philosophy makes it particularly ideal for projects with high demands on code size or customization flexibility.
