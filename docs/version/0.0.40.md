# v0.0.40 版本
该版本为组件库的0.0.40版本。在这个版本中，完成了表单组件的初步实现。并采用 `Vitest` 完成单元测试。该组件在 Vue 文件中实现，到此已经发布了40次，为早期的测试版本。当前版本表单依然存在一些潜在问题，所以为测试版本。也是表单的过度阶段。
