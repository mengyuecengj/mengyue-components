# v0.0.46 版本
该版本为组件库的0.0.46版本。在这个版本中，完成了表单组件的初步实现。并采用 `Vitest` 完成单元测试。该组件在 Vue 文件中实现，到此已经发布了46次，为早期的测试版本。当前版本完成了测试，当前为第一个表单可用版本。
