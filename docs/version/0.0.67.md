该版本为组件库的0.0.67版本。在这个版本中，完成了组件潜在问题的修复和vitest测试的100%覆盖。并采用 `Vitest` 完成单元测试。该组件在 Vue 文件中实现，到此已经发布了67次，为早期的测试版本的演进。目前规划，该版本为早期的最后一个版本。
