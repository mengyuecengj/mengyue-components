v0.0.68-beta.1 版本 ​

该版本为组件库的0.0.68-beta.1版本。在此版本中完成了测试版本演进的全部组件测试，为测试版本演进到正式版本的过度版本，为第一个正式版本发布前的第一个测试版，到次将组件库命名从mengyue-plus改成了mengyue-components，到此处已经发布了68次，正式版本的第一次发布。

安装组件库 ​

到此安装方式需要更改

bash
pnpm install mengyue-components
npm install mengyue-components
// 在项目的main导入
import MengyueComponents from 'mengyue-components'
import 'mengyue-components/style.css'

const app = createApp(App);
app.use(MengyueComponents)
app.mount('#app');
