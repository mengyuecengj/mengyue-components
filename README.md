## 🚀 快速开始

### 安装

# 使用 pnpm
pnpm add mengyue-plus

# 使用 npm
npm install mengyue-plus

# 使用 yarn
yarn add mengyue-plus

# 全局注册, 在项目的main中
```js
import { createApp } from 'vue'
import MengyuePlus from 'mengyue-plus'
import 'mengyue-plus/dist/style.css'

const app = createApp(App)
app.use(MengyuePlus)
app.mount('#app')
```
